Covid SwabTest

Clone project: - git clone https://github.com/Yogesh-YRM/Covid-Swabtest.git

create a new file ".env" and paste everything from ".env.example" inside it

import database

Config database: - .env

Migrate database: - php artisan migrate

Migrate refresh: - php artisan migrate:fresh

Create record in database a specified in code: - php artisan db:seed 

Run server: - php artisan serve
